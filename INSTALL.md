# インストール手順

## 1. プロジェクトのビルド

まず、依存関係をインストールしてプロジェクトをビルドします。

```bash
# 依存関係のインストール
npm install

# ビルド
npm run build
```

ビルドが完了すると、`dist` フォルダに拡張機能のファイルが生成されます。

## 2. アイコンの準備（オプション）

`public/icons/` フォルダに以下のサイズのアイコン画像を配置してください：

- `icon16.png` - 16x16ピクセル
- `icon48.png` - 48x48ピクセル
- `icon128.png` - 128x128ピクセル

アイコンがない場合でも拡張機能は動作しますが、デフォルトアイコンが表示されます。

## 3. Chromeへのインストール

### 手順

1. Chromeブラウザを開く
2. アドレスバーに `chrome://extensions/` と入力してEnterキーを押す
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
5. プロジェクトの `dist` フォルダを選択

### インストール後の確認

- ツールバー（ブラウザ右上）に拡張機能のアイコンが表示されます
- アイコンをクリックするとメモパネル（ページ右上のオーバーレイ）が開きます

## 4. 使い方

### 基本操作

1. **メモパネルを開く**
   - ツールバー（ブラウザ右上）のアイコンをクリック
   - または、`Cmd+Shift+M` (Mac) / `Ctrl+Shift+M` (Windows)

2. **メモを書く**
   - パネルが開いたらすぐに入力できます
   - 入力すると約800ms後に自動保存されます
   - 等幅フォントでコードやメモが読みやすい

3. **パネルのサイズ変更**
   - パネル左下のハンドルをドラッグ
   - デフォルトサイズ: 600x700ピクセル
   - 最大サイズ: 800x900ピクセル

### ショートカットキー

- **パネルを開く/閉じる**: `Cmd+Shift+M` (Mac) / `Ctrl+Shift+M` (Windows)

## 5. 開発時の注意事項

### ウォッチモード

開発中は以下のコマンドでウォッチモードを使用できます：

```bash
npm run dev
```

ソースコードを変更するたびに自動的にビルドされます。

### 拡張機能の再読み込み

ソースコードを変更した後は、`chrome://extensions/` ページで拡張機能の「再読み込み」ボタンをクリックする必要があります。

### デバッグ

- **Background script**: `chrome://extensions/` → 拡張機能の「Service Worker」リンクをクリック
- **Content script**: ページ上で右クリック → 検証 → Consoleタブ
- **Popup**: 現状は未使用（`public/manifest.json` に `default_popup` なし）

## トラブルシューティング

### パネルが表示されない

以下のページでは拡張機能が動作しません：

- `chrome://` で始まるページ
- Chrome Web Store
- その他のChrome内部ページ

### ショートカットキーが動作しない

1. `chrome://extensions/shortcuts` を開く
2. 「Chrome拡張メモアプリ」のショートカットを確認
3. 必要に応じてカスタマイズできます

### データが同期されない

- 本文は端末ローカルに保存されるため、同期されません
- フォルダ構造とメモのメタデータのみが同期されます
